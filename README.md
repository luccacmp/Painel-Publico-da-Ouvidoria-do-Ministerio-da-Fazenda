# Projeto Público da Ouvidoria do Ministério da Fazenda

Dashboard analítico desenvolvido em Power BI que centraliza e visualiza indicadores operacionais da Ouvidoria do Ministério da Fazenda em tempo real. O projeto integra dados de demandas, permitindo análise multidimensional por área, tempo de resposta, tipo de demanda e status de atendimento. A solução proporciona visibilidade operacional clara, facilitando a tomada de decisão gerencial e o acompanhamento de metas de desempenho da Ouvidoria.

## Sobre o projeto
Painel Publico da Ouvidoria do Ministério da Fazenda aonde estão disponibilizados dados como número de demandas por área, tempo médio de resposta, assunto, tipo de demanda, número de demandas respondidas, em andamento ou atrasadas.

## Tecnologias utilizadas
- Power BI - Desenvolvimento e publicação do dashboard
- Power Query - Extração e transformação de dados
- DAX - Cálculo de métricas e indicadores
- Modelagem de Dados - Estrutura relacional otimizada
- DataLake Serpro - Fonte de dados centralizada
